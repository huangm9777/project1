In this week, I learnt align-item; justify-content; flex-wrap. CSS align-item is to make the item in containers to be aligned to a crossing axes; justify-content is to help aligning items in the main axes; flex-warp make the content to be warped in row or column.
Source1: https://flexboxfroggy.com/
Example:
div {
display: flex;
align-items: center;
}
div {
display:flex;
flex-wrap: wrap;
}
